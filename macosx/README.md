# Mac OS X Configuration

Mac OS Xの設定をいい感じにするスクリプトたち．

## finder.sh

Finderとかファイルまわりの設定．やってることは大体こんな感じ：

* 最初にホームディレクトリを開く
* ウィンドウ名をフルパスに（ついでにステータスバーにも）
* 隠しファイルも表示する
* 拡張子の変更時に警告しない
* リスト表示をデフォルトにする
* USBなどの外部ストレージに`.DS_Store`を作らない
* 外部ストレージがマウントされた時にFinderで開く
