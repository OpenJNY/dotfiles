# LaTeX

## インストール

* MacTeX をインストールした．
* https://qiita.com/hideaki_polisci/items/3afd204449c6cdd995c9

#### Ghost-Script のインストール

```bash
brew install ghostscript
```

#### MacTeX のインストール

* [ミラーサイト](https://texwiki.texjp.org/?MacTeX#mirror)から最新版のMaXTeXをダウンロード
* 「install」のところで Customize を押して，ghostscript のインストールのチェックを外す

#### アップデート

```bash
sudo tlmgr update --self --all
```
