## 設定

#### Font: Hack, 14pt
* Powershell のパッチをあてたフォントを配布してるリポジトリがあるので，そこからインストール

> https://github.com/powerline/fonts

```bash
git clone https://github.com/powerline/fonts.git --depth=1
cd fonts
./install.sh
cd ..
rm -rf fonts
```
