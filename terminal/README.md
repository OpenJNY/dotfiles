# bash

シェルスクリプトの tips を集める

## Shebang (シェバン)

```sh
#!/usr/bin/env bash
```

## Set ()

### テンプレ

プラス記号を使って `+hoge` とするとフラグを折ることができる．
当然 `-hoge` で再度フラグを立てられるので，次のような使い方ができる．

```bash
#!/bin/bash

デバッグの必要ない処理

set -x #デバッグ出力開始

デバッグの必要な処理

set +x #デバッグ出力終了

デバッグの必要ない処理
```

### 説明

* `e` : エラーがあったら終了（`exit 0` 以外が返るものがあったら止まる）
* `u` : 未定義の変数を使ったら終了
* `n` : 文法のチェックのみを行う（実行はしない）
* `v` : 実行するコマンドを表示する
* `x` : デバッグ情報を表示する（変数の代入とか）
